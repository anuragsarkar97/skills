# Ops Skill Name

Use this skill when production behavior, reliability, or operational response matters.

## Workflow

1. Establish current state, impact, and recent changes.
2. Read the strongest signals first: errors, latency, traffic, saturation, logs, traces, deploys.
3. Form the top hypotheses and the evidence that confirms or rules each out.
4. Decide the safest action: rollback, hotfix, mitigation, or escalation.
5. Communicate status and document follow-up work.

## Output

State impact, evidence, hypothesis, next action, owner, and next update time.
